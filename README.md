# RevAir
Project 1 Team G

## Description
This web service allows users to create, modify and delete their own airline tickets, as well as the flights themselves. A custom ORM is used to persist the data.

## Tech Stack
  - Tomcat
  - Java 8
  - Custom ORM
  - Apache Maven
  - Amazon Web Services RDS
  - MariaDB
  - Git/Github
  - Postman
